# Rolisteam Documentation Website

## Get the code
```
$ git clone --recurse-submodules git@github.com:Rolisteam/Documentation.git
$ cd Documentation 
```

##Installation

```
$ mkvirtualenv roli
$ pip install -r requirements.txt
```

## Runing the dev server

```
$ pelican -lr
```

Now open http://localhost:8000 in your browser
